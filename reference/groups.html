Send a chat write a private chat message

Reason for existence

Who doesn't want to talk to their friends while playing a game?

Invocation form

POST /groups/[id]/messages

The [id] parameter should be available from the check-in call detailed above.

body
The body of the chat. Limited to 255 characters

Status

Group event check-in

Reason for existence

Users that are a member of groups will wish to both gauge group interest in an event at a glance, as well as know how the group competition is unfolding while in progress

Invocation form

The [group_id] parameter should be one available from the listing call detailed above.

The [live_event_id] parameter should be one available from the Live Event Check-in call.
