Summary

Data Usage

id
Used for unique identification of the selection. Currently not used.
chosen
Indicates the multiple choice selection you made
correct
Was this a correct answer to the question?

URLs

check
Can be used to update results of the prompt surrounding this selection (e.g. poll results, scoring status)
