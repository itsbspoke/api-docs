Summary

Data Usage

id
Live event id's uniquely identify an event numerically. This can be used in other API requests, (e.g. Live Event Check-In, one of the most important API calls we expose.
event_date
Start time of the event (US Eastern Time currently)
in_progress
Is the competition currently underway?
concluded
Has a winner been declared?
name
The display name for the event
tv_info
The network or broadcast entity where a player can find the source material for gameplay
demo
Demo events are only visible to beta testers; this flag indicates the events are for testing purposes.
