Questions

How could/should we indicate which answer is correct? In the choices array, higher up, or somewhere else?

Summary

Data Usage

id
Used for unique identification of the trivia question.
type
The literal string "Trivia"-- your should display Trivia like any other mulitple choice prompt.
body
Body of the question, should receive primary display.
commentary
Commentary on the question, can receive italicized or smaller display to taste.
tagline
Description of question type, usually indicates whether question will be scored.

Choices

submit
If the user selects this choice, send this answer when Answering a multiple-choice question.
display
Use this text to display to the user.

URLs

answer
Submit to this path to answer this question.
results
Issue a GET request to this path to get an answer breakdown
