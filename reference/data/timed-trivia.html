Timed Trivia Model

Question about the past with a time limit

JSON

{
 "question": {
  "type": "TimedTrivia",
  "id": 1,
  "correct": {
   "answer": "a",
   "selected": false,
   "correct": true,
   "display": "Singular of trivia"
  },
  "choices": [
   {
    "answer": "a",
    "selected": false,
    "correct": true,
    "display": "Singular of trivia"
   },
   {
    "answer": "b",
    "selected": false,
    "correct": false,
    "display": "Not a word"
   }
  ],
  "body": "Trivium",
  "client_id": "TimedTrivia_1",
  "commentary": null,
  "results": null,
  "urls": {
   "answer": "/timed_trivia/1/selections",
   "expiry": "/timed_trivia/1/expire"

 }
}

Available via

Weekly Game play API call.

Questions

Should we indicate which the correct answer is at all times?

Summary

Data Usage

id
Used for unique identification of the timed trivia. Currently used when answering.
type
The literal string "TimedTrivia"
body
Body of the question, should receive primary display.
commentary
Commentary on the question, can receive italicized or smaller display to taste.
answer_commentary
Commentary on the answer, display after the user has answered.

Choices

submit
If the user selects this choice, send this answer when Answering a multiple-choice question.
display
Use this text to display to the user.

URLs

answer
Submit to this path to answer this question.
expiry
Issue a POST request to this path to indicate time expired while waiting for a user answer
