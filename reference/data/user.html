Summary

This form is widely used in a variety of places. Be prepared to use it in many scenarios.

Data Usage

id
User id's uniquely identify a user numerically. This can be used in other API requests, (e.g. Group invitation.
avatar_url
An image chosen by the user to represent them during their gameplay.
name
How the user should be referred to during gameplay. May or may not be their real name.
level
The achieved level the user has received with their participation across multiple games.

Summary

This form contains unique user information gained only by logging in.

Data Usage

authentication_token
Use this in future request headers to identify yourself.
email
Passed back purely for reference, so that the user knows which account they are playing under.
id
May be referred to as user_id in this documentation, this number is used occasionally throughout the rest of the API.
