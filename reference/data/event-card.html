Data Usage

id
Used for unique identification of the event card. May not be used.
rookie
True if this is the first time the user has participated in an OtherScreen event.

Presence

status
active
away

Statistics

max_streak
The largest number of consecutive answers to receive the same score. Positive numbers indicate correct answers.
segment_score
Portion of the user's score occuring during the current event segment.
score
User's overall point total. Currently 10 points per correct answer.
longest_negative_streak
Largest number of questions answered incorrectly in a row during the event
longest_positive_streak
Largest number of questions answered correctly in a row during the event
accuracy
Formatted string indicating percentage of answers given correctly.
questions_answered
Number of scorable questions answered during the event

Summary

This form will most commonly be used to request your own event card during play. This allows you to study the details of your performance, and of your individual answers.

Data Usage

Check above section if the data element you are looking for is not present here.

Selections

pending
Answers to scorable questions for which the correct answer is not yet available.
scored
Answers to scorable questions that have been reflected in the card's score and statistics.
unscorable
Answers to unscorable questions, or scorable questions that have been marked as unscorable due to DJ error.
