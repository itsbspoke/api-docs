Summary

Data Usage

id
Weekly game id's uniquely identify an event numerically. This can be used in other API requests.
caption
Use this to display the name of the game
week_number
Number of the week within the year of this game
year
Year of contest
played_this_week
Has the authenticated user played yet this week?
questions[available]
Are there questions available to answer?
questions[available_count]
How many questions are available for me to answer?
