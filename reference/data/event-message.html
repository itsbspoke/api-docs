Event Message Model

Public chats

JSON

 {
  "event_message": {
   "id": 118600,
   "user": {
    "avatar_url": null,
    "id": 1,
    "name": "@gertig",
    "level": 1
   },
   "body": "What's up"
  }
 }

Available via

The Event chat history and Send a chat API endpoints. The Chat message WebSocket event will be the primary source.

Questions

None.

Summary

Data Usage

id
Used for unique identification of the chat message. Currently not be used.
body
Body of the chat message, should receive primary display.
