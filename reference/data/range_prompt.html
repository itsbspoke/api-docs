Summary

Data Usage

id
Used for unique identification of the prompt.
type
The literal string "RangePrompt"-- this indicates you should show the user a slider, dial, or other rangefinder.
body
Body of the question, should receive primary display.
commentary
Commentary on the question, can receive italicized or smaller display to taste.
tagline
Description of question type, usually indicates whether question will be scored.
min_point
The lowest value acceptable as an answer
max_point
The highest value acceptable as an answer
initial
The value the user should submit without moving the slider or rotating the dial.
step
How much each increment on the slider or notch on the dial should increment or decrement submitted value.

URLs

answer
Submit to this path to answer this question.
