Users

OtherScreen is people.

  • Profile

Profile what information is available about this player?

Reason for existence

Players will want to know about other people, whether they are friends or strangers.

Note This endpoint does not require authorization.

Invocation form

GET /users/[id]

[id] is a user's ID, available from a variety of locations.

Returns

A User Detail model.

Status

Complete

HTTP Error Codes

404
You must pass the user id for an existing user.
