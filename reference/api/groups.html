Group Listing What groups am I in? What groups can I join?

Reason for existence

To participate in groups, it's important to know which groups you are a part of, and which groups you have the opportunity to join.

Invocation form

GET /groups

Returns

An array of Group detail models.

Status

Complete

HTTP Error Codes

401
You must pass an authentication token.

Group chat history private chat history

Reason for existence

Players will have a variety of reasons to talk privately. Group chats last between events to create a long-running play supplement.

Invocation form

GET /groups/[id]/messages

The [id] parameter should be one available from the listing call detailed above.

Returns

An array of private chat messages.

Status

Warning This JSON form is definitely not correct. Don't use it.

HTTP Error Codes

401
You must pass an authentication token.
401
You must be a member of the group in question.
404
Make sure you are pulling the group id from the listing API call.

Send a chat write a private chat message

Reason for existence

Who doesn't want to talk to their friends while playing a game?

Invocation form

POST /groups/[id]/messages

The [id] parameter should be available from the check-in call detailed above.

body
The body of the chat. Limited to 255 characters

Returns

A private chat message.

Status

Complete

HTTP Error Codes

400
Ensure that you have supplied a chat body, and that it is less than 256 characters.
401
You must pass an authentication token.
401
You must be a member of the group in question.
404
Make sure you are pulling the event card id from the check-in API call.

Example usage

> curl -i -H "Accept: application/vnd.otherscreen.com; version=1" -H "HTTP_X_OTHERSCREEN_AUTH_TOKEN: ea71eb59a85972b9edecb4a14cbcde3eef19bd3242f5167b1d3e09560caba5a7" --data "body=Test message" http://localhost:5000/groups/47/messages
