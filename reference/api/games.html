Game Listing

Reason for existence

It returns a listing of games that the client will have an opportunity to play, both weekly and live. After authenticating, this should be the first request that clients make.

Invocation form

GET /events

Returns

An array of Live Event summary models, and an array of Weekly Game summary models

Status

Complete

HTTP Error Codes

401
You must pass an authentication token.
